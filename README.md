# 🐰✨ うさぎカレンダー - 夕飯記録アプリ

やさしいグリーンとうさぎモチーフの、可愛くて使いやすい夕飯記録アプリです。

## 🌟 特徴

- **可愛いデザイン**: セージグリーンとうさぎのイラストで癒しの雰囲気
- **紙の手帳風**: おしゃれでミニマルなUI
- **簡単記録**: タップするだけでメニュー、写真、メモを記録
- **検索機能**: 過去のメニューを簡単に検索
- **ローカル保存**: データはブラウザに保存され、オフラインでも使用可能
- **スマホ最適化**: iPhoneでの使いやすさを重視

## 📱 使い方

1. **記録する**
   - カレンダーの日付をタップ
   - メニュー名を入力
   - 写真を追加（任意）
   - メモを追加（任意）
   - 「保存」をタップ

2. **見る**
   - 記録した日付をタップで詳細表示
   - 写真、メニュー名、メモが確認できます

3. **編集・削除**
   - 詳細画面から「編集」または「削除」ボタン

4. **検索**
   - 上部の検索バーにメニュー名を入力
   - 過去の記録から部分一致で検索

## 🚀 デプロイ方法

### GitHub Pagesでデプロイ

1. GitHubで新しいリポジトリを作成
2. ファイルをすべてアップロード
3. Settings > Pages > Source で `main` ブランチを選択
4. 数分後にURLが発行されます

### Vercelでデプロイ

1. [Vercel](https://vercel.com)にログイン
2. 「New Project」をクリック
3. GitHubリポジトリを接続 or ファイルをドラッグ&ドロップ
4. 「Deploy」をクリック
5. 即座にURLが発行されます

### Netlifyでデプロイ

1. [Netlify](https://netlify.com)にログイン
2. フォルダをドラッグ&ドロップ
3. 自動的にデプロイされURLが発行されます

## 🛠️ 技術スタック

- **React 18** (CDN版)
- **Vanilla CSS** (カスタムスタイル)
- **LocalStorage** (データ永続化)
- **SVG** (うさぎイラスト)

## 💾 データ保存

- すべてのデータはブラウザのLocalStorageに保存されます
- サーバーは不要で、オフラインでも動作します
- ブラウザのデータを消去すると記録も消えるので注意してください

## 🎨 カスタマイズ

### 色を変更する

`css/style.css` の以下の部分を編集:

```css
/* メインカラー */
#b5c9b6  /* セージグリーン */
#fff8f0  /* クリーム */
#f5e6d3  /* ベージュ */
```

### うさぎイラストを変更する

`js/bunny-icons.js` でSVGを編集できます。
